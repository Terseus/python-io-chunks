io-chunks
#########

Stream chunks compatible with IO standard library


Installing
==========

.. code-block:: bash

    $ pip install io-chunks


Python version
==============

Tested in Python 2.7 and 3.2 to 3.6


Running the tests
=================

Run the tests with ``nose``:

.. code-block:: bash

    $ nosetests
