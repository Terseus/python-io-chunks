io-chunks
#########

Stream chunks compatible with IO standard library


Installing
==========

.. code-block:: bash
    pip install io-chunks


Python version
==============

Tested in Python 3.5.


Running the tests
=================

Run the tests with ``nose``:

.. code-block:: bash
    nosetests
